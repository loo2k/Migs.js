## 介绍

此版本的 Migs 是使用 **Node.js** 和 **Mongodb** 开发的一个图片管理程序，来源于 @LOO2K 开发的 PHP 版本 Migs [地址](loo2k.com/blog/sae-migs/)，并以移植项目来学习 Node.js 的使用，代码仅供研究，不建议实际使用；
